## 🧠 Key Learnings
- 💡 Hook quality determines if anyone watches the ad
- 🎯 Platform-specific scripts perform better than
  one-size-fits-all content
- 😩 Problem-focused opening creates more relatability
- ⭐ Social proof increases conversion significantly
- ⏰ Urgency and FOMO drive immediate purchases
